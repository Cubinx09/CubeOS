CODE NAME S.T.E.A.M.

Alien invasion in a steampunk world!

This game is just straight up strange. Of course there's stranger stuff out there but this game geniunely has me losing my mind.

Code Name S.T.E.A.M. is developed by Intellegent Systems (AKA the Fire Emblem guys), and released in 2015 on the 3DS.

Ok, so, in a steampunk alternative reality of the 1860s, an alien invasion strikes Earth. First, Britain. Main character, Henry Fleming, gets recruited by the secret organazation, S.T.E.A.M., to get rid of the evasion. Along the way, the organazation gets bigger, and with the aliens getting more threatening.

I haven't finished playing this game (I'm only on mission 5), but I just want to let you know that in this universe Lincoln is both alive and the leader of S.T.E.A.M. (one of his first lines is LITERALLY, "despite the rumors", as a response to Henry acting surprised that he is not dead, in this universe, Lincoln's assassination is a cover-up), an anthormorphosized lion is just an accepted part of society, and Tom Sawyer is in this game too.

Lincoln at some point pilots a mecha. It's crazy.

Gameplay wise, the game is a more forgiving strategy RPG game. It's fully in 3D, with the camera acting more like an over the shoulder shooter, meaning you have to fully move around the maps to traverse. There are not "troops" in this game. Every class of character (tank, healer, scout, etc) is just one person, so battles aren't as huge either. An interesting thing Code Name S.T.E.A.M. does is that for most weapons you have to aim to do damage, it is partially a third person shooter.

The music goes crazy, its very bombastic, very triumphant, but it knows to slow down once it's your turn to think about what to do next.

Code Name S.T.E.A.M. is virtually unknown which is very sad, because it's a fun game with loads of charm, more people should check it out, ESPECIALLY since online prices are crazy cheap, although the two times I bought a catridge of this game, one was corrupted, the other couldn't be read, so if you want to avoid that uhh.. pirate it. In my personal opinion I think it's better to play the game completely blind.

SPOLOUS EX

Arcade shoot em' up in the browser! (or as .exe if you prefer)

SPOLOUS EX is a side scrolling shoot em' up that's inspired by Gradius, with many similarities in both.

If on the title screen for too long, the game proceeds to give backstory on the game. Which summarized: In a galaxy far away from earth, an ore with the ability to create infinite energy was found; Spolum. One planet discovered this phenomena, and revolutionized its industry with the power the ore holds. Said planet then conquers the planet where Spolum is found, which upsets the other neighboring planets, and as a result, war commences and it ends with one of those planets completely barricading the planet that holds all the Spolum with the levels in the game. It's up to the person piloting the Estaprick, to stop all the fighting.

Despite being very similar to Gradius, SPOLOUS EX does have a stand out feature: the spaceship types.

There are over a dozen or so "types" that you can choose from before entering a level in SPOLOUS EX. These types give the spaceship's weapons different capabilities, which changes playstyle slightly. One type can have the missiles shoot in a forward direction, another can go off to the side. Another type can have boosted bullets, while another has a laser. Some types provide a shield, others don't.

You start off with quite a few types to choose from, but you have to unlock the rest with credits, the games currency.

By pressing the C key (if on keyboard), you can change the directory of your multiples (which act like your clones in a sense), but the way the trajectory changes depends on which type to use.

Gameplay is hard but in a satisfying way. Whenever you lose a run an overwhelming sense of frustration doesn't really wash over, unless you been playing the game for hours on end of course, and repeating sections of levels doesn't get tiring either.

Music and sound effects are in my opinion a stand out for this game, as the tunes are nice and all the effects present in the game are all punchy and effective. The music has all sorts of synths and a more 'futuristic' vibe to them.

The creator of the game, suppon, is Japanese, so the english translations in the game are all wonky but serviceable. "Easy start" is really the 'easy' mode of the game, and the game's description doesn't exactly make it clear what the keyboard controls are so I'll list them down below.

Overall, SPOLOUS EX is a fun game that's worth your time, especially since it's free!

Controls (on keyboard)

  • X: fire
  • Z: power-up
  • C: use special
  • Tab: toggle menu
  • arrow keys: movement (obviously)
